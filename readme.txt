1. Upload `easy-repeater.zip` to the `/wp-content/plugins/` directory
OR search for "easy repeater" in wordpress plugins . 
2. extract it .
3. Activate the plugin through the 'Plugins' menu in WordPress .
4. see documentaion in the menu .
